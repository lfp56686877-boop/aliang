## 二、AI自媒体全流程

### 第一步：AI选题（15分钟）

用AI找到热门话题：

```
请帮我找10个关于[领域]的自媒体选题：
要求：
1. 结合最近一周的热点
2. 目标读者是[人群]
3. 有争议性或实用价值
4. 标题要吸引点击
```

**进阶技巧**：
- 用百度指数/微信指数验证热度
- 用AI分析竞品的热门内容
- 建立选题库，持续积累

### 第二步：AI写稿（30分钟）

**大纲生成**：
```
请为以下选题生成一份文章大纲：
选题：[具体选题]
字数：1500字
结构：引言+3个核心观点+总结
```

**逐段生成**：
```
请根据以下大纲，写第一部分的正文：
[粘贴大纲第一部分]
要求：用数据引入，150字以内
```

**润色优化**：
```
请优化以下文章：
1. 增加案例
2. 语气更轻松
3. 删掉冗余内容
```

### 第三步：AI配图（10分钟）

**方案A：AI生成原创图**
- 用Midjourney/通义万相生成文章配图
- 保持风格一致

**方案B：AI推荐素材**
- 用AI描述需求，推荐合适的素材图
- 从Pexels/Pixabay下载免费素材

### 第四步：AI排版（5分钟）

- 用135编辑器/秀米的AI排版功能
- 一键美化排版
- 添加品牌元素

### 第五步：AI标题优化（5分钟）

```
请为以下文章生成5个标题选项：
文章主题：[主题]
要求：有数据、有悬念、有行动号召
```

## 三、不同平台的AI运营策略

### 公众号
- **内容深度**：1500-3000字深度文章
- **AI重点**：内容质量和结构
- **更新频率**：每周2-3篇

### 小红书
- **内容特点**：图文并茂，实用干货
- **AI重点**：标题和封面图
- **更新频率**：每天1-2条

### 抖音
- **内容形式**：短视频
- **AI重点**：脚本和画面
- **更新频率**：每天1-2条

### B站
- **内容深度**：长视频深度内容
- **AI重点**：脚本和剪辑
- **更新频率**：每周1-2条

### 知乎
- **内容特点**：专业深度回答
- **AI重点**：逻辑性和专业性
- **更新频率**：每天1-2个回答

## 五、常见问题

**Q：AI写的内容有灵魂吗？**
A：AI生成的是初稿，你需要注入个人观点和经验。AI是工具，灵魂靠你。

**Q：会不会被平台限流？**
A：平台关注的是内容质量，不是生产方式。关键是内容有价值。

**Q：需要多少时间？**
A：每天1-2小时足够。AI让你把时间花在"思考"而不是"写作"上。
