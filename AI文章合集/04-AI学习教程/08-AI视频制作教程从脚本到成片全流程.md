## 二、全流程教程

### 第一步：脚本创作（5分钟）

用AI生成视频脚本：

```
请为以下主题写一个60秒的短视频脚本：
主题：为什么你应该开始使用AI工具
要求：
1. 开头3秒要抓人（用一个反问或数据）
2. 中间讲3个核心观点，每个15秒
3. 结尾给出行动号召
4. 标注每个画面的文字描述
5. 预估每段的配音时长
```

**AI输出示例**：
```
【画面1：黑屏+大字】
"你知道吗？90%的人还没用过AI工具。"
时长：3秒

【画面2：AI工具界面截图】
"但用过的人，效率提升了3倍。"
时长：5秒

...（省略）
```

### 第二步：AI配音（5分钟）

**方案A：剪映（推荐）**
1. 打开剪映，新建项目
2. 点击"文本朗读"
3. 粘贴脚本文字
4. 选择音色（推荐：阳光男声/温柔女声）
5. 生成语音

**方案B：ElevenLabs（更自然）**
1. 访问 elevenlabs.io
2. 输入脚本文字
3. 选择音色
4. 生成并下载音频

### 第三步：画面生成（15分钟）

**方法1：AI图片+Ken Burns效果**
1. 用Midjourney/通义万相生成与脚本匹配的图片
2. 在剪映中导入图片
3. 添加Ken Burns效果（缩放平移），让静态图片动起来

**方法2：AI视频生成**
1. 用可灵/Sora根据脚本描述生成视频片段
2. 每个片段5-10秒
3. 导入剪映拼接

**方法3：素材库+AI剪辑**
1. 用Pexels/Pixabay下载免费素材
2. 剪映AI自动匹配脚本和素材

### 第四步：视频剪辑（10分钟）

1. **导入素材**：将配音、图片/视频导入剪映
2. **按脚本排列**：按照脚本顺序排列素材
3. **调整时长**：让画面时长与配音匹配
4. **添加转场**：使用剪映的转场效果
5. **添加字幕**：剪映自动生成字幕

### 第五步：后期优化（5分钟）

1. **背景音乐**：剪映内置音乐库，选择合适的BGM
2. **调色**：使用剪映的滤镜统一色调
3. **音量调整**：确保配音清晰，BGM不抢戏
4. **导出**：选择1080p，导出视频

## 三、不同场景的制作方案

### 知识科普类
- 脚本：ChatGPT生成
- 画面：AI图片+文字动画
- 配音：剪映AI语音
- 时长：1-3分钟

### 产品介绍类
- 脚本：ChatGPT生成
- 画面：产品实拍+AI特效
- 配音：真人录制或AI语音
- 时长：30秒-1分钟

### 故事叙述类
- 脚本：ChatGPT生成
- 画面：AI图片+转场动画
- 配音：AI语音+背景音乐
- 时长：1-5分钟

### Vlog类
- 脚本：AI辅助大纲
- 画面：手机实拍
- 配音：真人录制
- 时长：3-10分钟

## 四、效率提升技巧

1. **建立模板**：为常用视频类型建立剪映模板
2. **批量生成**：一次生成多个视频脚本，批量制作
3. **素材库**：建立常用的AI图片和视频素材库
4. **复用内容**：将一篇文章转化为多个短视频

## 五、常见问题

**Q：AI生成的视频有版权问题吗？**
A：使用AI工具生成的内容，版权政策各不相同。商用前请检查平台条款。

**Q：如何提高视频质量？**
A：脚本质量决定视频质量。先花时间打磨脚本。

**Q：适合什么平台？**
A：抖音、B站、小红书、视频号都可以。不同平台调整时长和比例。

## 六、总结

AI视频制作的核心流程：脚本 → 配音 → 画面 → 剪辑 → 后期。整个流程1小时内可以完成。关键是脚本质量，画面是锦上添花。

从今天开始，把你手头的内容转化为视频，用AI降低制作门槛。
