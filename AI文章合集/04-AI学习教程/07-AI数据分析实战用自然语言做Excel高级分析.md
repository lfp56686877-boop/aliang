## 三、5个实战案例

### 案例1：销售数据分析

**数据**：一份10万行的销售记录CSV

**提示词**：
```
请分析这份销售数据：
1. 按月统计销售额趋势
2. 找出销售额TOP10的商品
3. 分析各区域的销售占比
4. 识别销售异常波动并分析原因
5. 生成可视化图表
```

**AI输出**：自动生成4张图表+分析报告，发现某区域Q3销售额异常下降30%，原因是主要经销商更换。

### 案例2：用户行为分析

**数据**：电商用户行为日志

**提示词**：
```
分析用户行为数据：
1. 计算用户留存率（次日、7日、30日）
2. 识别高价值用户特征
3. 分析用户流失原因
4. 提出提升留存的建议
```

### 案例3：财务数据分析

**数据**：公司季度财务报表

**提示词**：
```
分析这份财务数据：
1. 计算关键财务指标（毛利率、净利率、ROE）
2. 与去年同期对比
3. 分析成本结构变化
4. 预测下季度趋势
```

### 案例4：市场调研分析

**数据**：问卷调查结果

**提示词**：
```
分析这份问卷数据（500份）：
1. 计算各问题的选择分布
2. 分析不同人群的回答差异
3. 识别关键洞察
4. 生成可视化报告
```

### 案例5：A/B测试分析

**数据**：A/B测试结果

**提示词**：
```
分析这次A/B测试：
1. 计算两组的转化率
2. 进行统计显著性检验
3. 判断哪个版本更优
4. 计算需要多少样本量才能得出可靠结论
```

## 四、数据分析提示词模板

### 探索性分析
```
请对这份数据进行探索性分析：
1. 数据概览（行数、列数、数据类型）
2. 缺失值和异常值检查
3. 关键统计指标
4. 可视化展示
```

### 对比分析
```
请对比[A]和[B]：
1. 核心指标对比
2. 差异显著性检验
3. 差异原因分析
4. 建议和结论
```

### 趋势分析
```
请分析以下时间序列数据：
1. 整体趋势
2. 季节性规律
3. 异常点识别
4. 未来预测
```

## 五、提高分析质量的技巧

1. **明确分析目标**：告诉AI你想解决什么问题
2. **提供背景信息**：让AI了解业务上下文
3. **分步骤提问**：复杂分析拆解为多轮对话
4. **验证结果**：关键数据需要人工复核
5. **追问细节**：发现异常时让AI深入分析

## 六、常见问题

**Q：数据安全吗？**
A：上传到ChatGPT的数据会被OpenAI处理。敏感数据建议使用本地工具或脱敏处理。

**Q：分析结果准确吗？**
A：AI分析结果需要人工验证。关键决策不能完全依赖AI。

**Q：处理大文件很慢？**
A：ChatGPT有文件大小限制。大文件建议先用Excel筛选再上传。
