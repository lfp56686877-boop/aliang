# Prompt工程入门：如何写出高质量的AI提示词

> 同样的AI模型，不同的提示词可以产出天壤之别的结果。Prompt工程是AI时代最实用的技能之一。本文系统讲解提示词的写作方法，让你的AI输出质量提升10倍。

## 一、为什么提示词重要

AI模型就像一个超级聪明但"不懂读心术"的助手。你说得越清楚，它做得越好。

**同一个模型，不同提示词的效果对比**：

❌ 差的提示词：
```
写一篇文章
```
→ AI不知道写什么主题、给谁看、什么风格、多长

✅ 好的提示词：
```
你是一个科技博主，读者是25-35岁的互联网从业者。请写一篇1500字的文章，主题是"为什么每个人都应该学习使用AI"。要求：开头用数据引入，正文分3个论点，每个论点配一个真实案例，结尾给出行动建议。语气要轻松但有说服力。
```
→ AI输出质量天差地别

## 二、提示词的核心公式

### 基础公式（适用于80%的场景）
```
角色 + 任务 + 上下文 + 格式 + 约束
```

**示例**：
```
角色：你是一个资深HR
任务：帮我写一份产品经理的JD
上下文：初创公司，B轮，需要能独立负责产品的全能型人才
格式：包含职位描述、任职要求、加分项
约束：不要超过500字，不要用"抗压能力强"等套话
```

### 进阶公式（适用于复杂任务）
```
角色 + 背景 + 目标 + 任务 + 示例 + 格式 + 约束 + 输出检查
```

**示例**：
```
角色：你是一个10年经验的数据分析师
背景：我有一份电商销售数据（CSV），包含日期、商品、销量、金额字段
目标：找出销售趋势和异常点，为下季度备货提供决策支持
任务：
1. 分析月度销售趋势
2. 识别销售TOP10商品
3. 发现异常波动并分析原因
示例：输出格式参考下方表格
格式：使用Markdown表格+文字分析
约束：不要用技术术语，面向非技术人员
输出检查：确保每个结论都有数据支撑
```

## 三、六大提示词技巧

### 1. 角色设定
让AI扮演特定角色，输出会更专业：

```
你是一个拥有20年经验的资深律师
你是一个从小红书粉丝50万的博主
你是一个在Google工作过的高级工程师
```

### 2. Few-shot（给示例）
给AI一个输出样例，它会模仿：

```
请为以下产品写标题，参考示例的风格：
产品：智能降噪耳机
示例标题：「地铁上的私人音乐会」——这款耳机让我在嘈杂世界找到宁静

产品：便携榨汁杯
标题：
```

### 3. 链式思考（Chain of Thought）
让AI展示推理过程：

```
请一步一步分析这个商业问题：
1. 首先列出问题的关键因素
2. 然后分析每个因素的影响
3. 接着评估各方案的优劣
4. 最后给出建议
```

### 4. 约束条件
明确告诉AI什么不要做：

```
要求：
- 不要使用"赋能"、"抓手"等互联网黑话
- 不要超过800字
- 不要编造数据
- 不要使用"首先、其次、最后"等模板化表达
```

### 5. 输出格式
指定输出格式让结果更可用：

```
请用以下格式输出：
## 标题
### 要点一
（200字说明）
### 要点二
（200字说明）
### 总结
（100字总结）
```

### 6. 迭代优化
第一轮输出不满意？追问优化：

```
请把上面的内容：
1. 语气改得更轻松幽默
2. 增加一个真实案例
3. 把第三段删掉，太啰嗦
```

## 四、常见场景提示词模板

### 写作
```
你是一个[领域]的资深写手。请为[目标读者]写一篇[字数]的[文体]，主题是[主题]。要求：[具体要求]
```

### 分析
```
你是一个[领域]的专家分析师。请分析[分析对象]，从[维度1]、[维度2]、[维度3]三个维度展开。输出格式：[格式]
```

### 翻译
```
你是一个专业翻译。请将以下内容翻译为[语言]，要求：[风格要求，如"商务正式"/"口语化"/"文学性"]
```

### 编程
```
你是一个[语言]高级开发者。请用[语言]实现[功能]。要求：[技术要求]
```

### 创意
```
你是一个创意总监。请想出[N]个[类型]的创意，要求：[约束条件]
```

## 五、提示词进阶：系统提示词

系统提示词可以设定AI的"基础人格"，让每次对话都继承设定：

```
# 系统提示词
你是一个专业的科技内容创作者。你的风格是：
1. 用数据说话，不空谈
2. 语言简洁，避免废话
3. 给出具体可执行的建议
4. 用比喻和类比让复杂概念易懂
5. 保持客观，不盲目吹捧
```

## 六、总结

提示词不是"玄学"，而是"沟通技巧"。核心原则：
1. **清晰**：明确告诉AI你要什么
2. **具体**：给出具体的约束和要求
3. **迭代**：不满意就追问，AI不怕麻烦
4. **练习**：写得越多，效果越好

从今天开始，把你每天重复的工作用AI完成，每次练习写更好的提示词。一个月后，你会发现自己的AI使用能力已经远超90%的人。
